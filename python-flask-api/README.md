## Installing Flask

Flask work with a instalation like python and with this you can create the project

We can install pip from `pip`, to install `pip`:

mac:

```
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
```

to install:

```
python3 get-pip.py
```

to run:

```
pip3.9 install <library>
```
